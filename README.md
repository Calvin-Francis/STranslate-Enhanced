## ✨ 新增功能

### 屏幕覆盖翻译

截图选区后，翻译结果**直接覆盖显示**在原文位置，无需切换窗口。

> 💡 **非常适合桌面 APP 内翻译**——不再需要麻烦地截图后打开翻译器，也不用拿起手机拍照翻译，一键搞定！

| 功能 | 说明 |
|------|------|
| **原地翻译** | 翻译文本直接覆盖原文显示 |
| **可交互文本块** | 支持拖拽移动、Ctrl+滚轮缩放 |
| **双击编辑** | 双击文本块可编辑、复制内容 |
| **智能段落合并** | 自动识别并合并同一行的文本 |
| **半透明背景** | 90% 透明背景，不遮挡视线 |

### 其他改进

- 基于 **Fluent Design 2** 重构界面
- 支持 **.NET 10** 框架
- 引入**插件系统**，可扩展 OCR 和翻译服务

## 🎯 使用方法

1. 启动程序后，使用快捷键Alt+S触发**截图翻译**
2. 框选需要翻译的屏幕区域
3. 等待 OCR 识别和翻译完成
4. 翻译结果自动覆盖显示在原位置
5. 可拖拽、缩放、编辑文本块
6. 按 `Esc` 关闭覆盖层
